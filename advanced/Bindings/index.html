ProFTPD Developer's Guide: The Bindings API

ProFTPD Version 1.2

Table of Contents

The Bindings API is an abstraction layer for dealing with the association between a server configuration information and the address at which clients encounter that server. This association is codified by an object called a binding in ProFTPD.

A developer wishing to make use of the capabilities of bindings should read the description of the binding objects first, then the functions for working with these objects.
