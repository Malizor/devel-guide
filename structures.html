Module Structures
The structures discussed here provide the fundamental building and operating units for ProFTPD modules.

MODRET
The MODRET is a #define alias for a module_struc, and is the main message unit that is returned by most module handlers (authentication handlers are a little different).

config_rec
A typedef of config_struc, this structure is the storage unit used to maintain the configured options and states of the ProFTPD engine.

cmd_rec
This data type is a typedef of cmd_rec, and is the data "message" passed to configuration directive and to FTP command handlers.

server_rec
This structure holds most of the main configuration data for a ProFTPD server, be it the "main" server or a <VirtualHost>.
